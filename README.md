## Development

After clone this repo...
```sh
npm install
npm start
```

## With docker

Build and start container
```sh
docker-compose up -d --build
```

Stop the container
```sh
docker-compose stop
```

## Production
```sh
npm run build
```

and serve with any server (http-server or configure nginx or whatever)
